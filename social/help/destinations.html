送信先

YouTube、Facebook、Twitterに動画を公開する宛先を作成します。

YouTubeの宛先を作成する

YouTubeの宛先を作成するには、次の手順を実行します。

  1. クリック目的地を追加.
  2. クリックYouTube でログイン.
  3. YouTubeチャネルに関連付けられているGoogleアカウントにログインします。
  4. YouTube アカウントを管理する権限を要求するダイアログが開きます。
  5. クリック許可する.
  6. 宛先に一意のわかりやすい名前を付けます。クリック追加.
  7. を参照してくださいYouTube のデフォルトの設定デフォルト値の設定については、セクションを参照してください。
  8. クリック行き先ページを更新します。宛先名の下の値は、宛先 ID です。

詳細はこちら

Facebookの宛先を作成する

Facebookの宛先を作成するには、次の手順を実行します。

  1. クリック目的地を追加.
  2. クリックFacebookでサインイン.
  3. Facebookの資格情報を入力して、ログインするあなたのアカウントに。
  4. クリック継続するBrightcove Social があなたのプロフィール情報を受け取ることを確認します。
  5. クリックわかったBrightcove Social がページを管理および公開し、ページとアプリのインサイトにアクセスできることを確認します。
  6. 宛先に一意のわかりやすい名前を付けます。クリック追加.
  7. を参照してくださいFacebook のデフォルト設定デフォルト値の設定については、セクションを参照してください。
  8. クリック行き先ページを更新します。宛先名の下の値は、宛先 ID です。

詳細はこちら

Twitterの宛先を作成する

Twitterの宛先を作成するには、次の手順を実行します。

  1. クリック目的地を追加.
  2. 選択するTwitterでログイン.
  3. 動画を投稿したいTwitterアカウントにログインします。クリックアプリの承認.
  4. 宛先に一意のわかりやすい名前を付けます。クリック追加.
  5. を参照してくださいTwitter のデフォルト設定デフォルト値の設定については、セクションを参照してください。
  6. クリック行き先ページを更新します。宛先名の下の値は、宛先 ID です。

詳細はこちら

宛先の削除

宛先を削除するには、次の手順を実行します。

  1. 左側のナビゲーションで、行き先現在の目的地のリストを表示します。
  2. 削除する宛先名をクリックします。
  3. 宛先の詳細で、 そしてクリック消去削除を確認します。
    4. 注:宛先を使用する自動同期ジョブが作成されている場合は、削除アイコンは表示されません。宛先を削除する前に、自動同期ジョブを削除する必要があります。

詳細はこちら

宛先に再接続する
ブライトコーブの権限がソーシャル メディア アカウントから削除された場合、または宛先の資格情報が無効になった場合、エラー メッセージが表示されることがあります。

問題を解決するには、次の手順に従います。

  1. 宛先名をクリックします。
  2. クリック再接続ページの上部にあるリンク。
  3. ソーシャルメディアアカウントで再認証を求められる場合があります。
