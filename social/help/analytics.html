ソーシャル·アナリティクス

ビデオのソーシャルパフォーマンスを監視します。

ダッシュボード

ソーシャルメディアアカウントのビデオパフォーマンスの概要を表示します。

  1. 時間範囲を選択します。
  2. クリック申し込みレポートを更新します。
  3. 動画再生回数の概要- 一定期間のソーシャル メディア アカウントからの動画再生回数
    • マイ サイトとアプリ- ビデオの再生開始時にカウントされるビデオ ビュー。 Video Cloud Analytics パフォーマンス レポートを開く
    • フェイスブック- ビデオが 3 秒以上視聴された場合にカウントされるビデオ ビュー
    • ユーチューブ- ビデオが約30秒間視聴された場合にカウントされるビデオビュー
  4. 人気の目的地- 一定期間のビューが最も多いトップ 5 の目的地
  5. トップ動画- 一定期間の再生回数が最も多いトップ 5 の動画;ビデオのタイトルをクリックして、ソーシャル ビュー分析を開きます
  6. 社会的見解- 一定期間、ソーシャル メディア アカウントに投稿されたビデオのビデオ ビュー。グラフの線にカーソルを合わせると、その日付のデータが表示されます

詳細はこちら

ソーシャルビュー

ソーシャルメディアアカウントに一定期間投稿されたビデオのビデオビューを表示します。

  1. 時間範囲を選択します。
  2. クリック申し込みレポートを更新します。
  3. 社会的見解- 一定期間、ソーシャル メディア アカウントに投稿されたビデオのビデオ ビュー。通常、ソーシャル プラットフォームが API のビューとエンゲージメント データを更新する前に、24 ～ 48 時間以上の遅延があります。
    • マイ サイトとアプリ- Video Cloud プレーヤおよびアプリからのビデオ ビュー
    • フェイスブック- Facebook での動画再生
    • ユーチューブ- YouTube での動画視聴
    • 合計- すべてのプラットフォームでの動画再生の合計数
  4. ソーシャル ビューの詳細- ビデオビューの詳細再生回数、平均視聴時間、平均視聴率.
  5. データのエクスポート- レポート データを CSV または Excel ファイルとしてダウンロード

詳細はこちら

エンゲージメント

ソーシャルメディアアカウントに共有されるビデオのソーシャルエンゲージメント（いいね、共有、リツイートなど）を表示します。

  1. 時間範囲を選択します。
  2. クリック申し込みレポートを更新します。
  3. ソーシャルエンゲージメント- 一定期間、ソーシャル メディア アカウントに投稿された動画に対するユーザー エンゲージメント。通常、ソーシャル プラットフォームが API のビューとエンゲージメント データを更新する前に、24 ～ 48 時間以上の遅延があります。
    • ツイッター- リツイート数とお気に入り数
    • フェイスブック- いいね、コメント、シェアの数
    • ユーチューブ- コメント、いいね、嫌い、シェアの数
  4. 動画によるソーシャル エンゲージメント- 各動画の詳細ビュー データ;列セレクターを使用する 表示するカラムを選択する
    各動画について以下が表示されます。
    • 動画の名前と再生時間
    • 動画ID
    • 作成日
    • 選択した列
  5. データのエクスポート- レポート データを CSV または Excel ファイルとしてダウンロード

詳細はこちら
