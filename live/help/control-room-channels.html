の制御室ページは、ライブ チャネルに関する情報を表示/編集するために使用されます。

ストリーミングの開始

ビデオストリームをエンコーダに接続するには、次の手順を実行します。

  1. クリック活性化ボタン。ストリーミングを開始する前に、チャネルをアクティブにする必要があります。
  2. エンコーダを開いてストリーミングを開始します。
  3. クリックコピーの中にストリーミング エンドポイント (RTMP URL)エンコーダーに貼り付けます。
  4. クリックコピーの中にストリーム名エンコーダーに貼り付けます。
  5. この情報を使用してエンコーダでストリーミングを開始すると、数秒後にビデオがControlRoomに表示されます。
  6. チャネルを停止するには、 をクリックします。ストリームをキャンセル.

ライブチャネル情報

の制御室ページには 5 つの主要なセクションが表示されます。

  • チャンネル情報- イベントに関する情報を次のように表示しますイベントステータス、現在の視聴者、 と経過時間.
  • チャンネルの詳細- チャネルの参照詳細を次のように表示しますチャンネル名、動画 ID 、 と簡単な説明.
  • チャネル構成設定- イベントの構成設定に関する情報を次のように表示します。ジョブID 、動画 ID 、状態と終点.
  • モニター- 現在のイベントからデータとグラフを取得します。
  • レンディション- 作成されたレンディションの詳細を表示します。レンディションのリンクをクリックすると、レンディションのエンコーディング設定が表示されます。
  • クリップ- ライブ ストリームから作成されたすべてのクリップを表示します。
