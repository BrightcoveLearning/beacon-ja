概要

再生リストは、Brightcove Beaconの2つの方法のいずれかで作成されたビデオおよびその他のエンティティのコレクションです。

  1. VideoCloudライブラリからBrightcoveビーコンにインポートされます。この場合、再生リストのアイテムは動画のみになります。
  2. を使用して Brightcove Beacon で作成されたプレイリストタブ。Brightcove Beaconで再生リストを作成するときは、Video Cloudからインポートしたビデオ（映画/エピソード）を使用できます。さらに、シリーズ、シーズン、チャンネル、ライブイベントを追加することもできます。この方法がこのドキュメントの焦点です。
次の点にご注意ください。
  • Brightcoveは、ビデオクラウドでビデオのみを含むプレイリストを作成することを推奨しています。
  • に示されているすべての機能ではありません。プレイリストタブはこの時点で使用できます。このドキュメントでは、どのフォーム項目に入力する必要があるかが明確になっています。

基礎データセクション

Brightcove Beaconでプレイリストを作成するには、次の手順に従います。

  1. Brightcoveビーコンにログインします。
  2. 上部のナビゲーションで、プレイリスト.

  3. クリック新しいプレイリストを追加ボタン。

  4. から基本データタブで、プレイリストの名前と表示タイプを定義し、コンテンツを追加します。このセクションのほとんどの機能は、現時点では使用できないことに注意してください。もちろん、以下の手順では、使用可能なアイテムのみを扱います。

  5. の中に名前フィールドに、プレイリストの内部システム名を追加します。これは、分析に使用される名前でもあります。後の手順で、プレイリストの表示可能なタイトルを追加することに注意してください。
    Brightcove Beacon vs. VideoCloudで作成されたプレイリストを何らかの方法で指定することをお勧めします。たとえば、アスタリスク ( * ) は、プレイリストが Brightcove Beacon で作成されるときに、プレイリスト名の前に付けられます。もちろん、アスタリスクである必要はありませんが、プレイリストを更新するときに Brightcove Beacon に移動するように、何らかの表記を選択してください。プレイリストタブまたは Video Cloud Studio で変更を加えます。
  6. からレイアウトを選択すべてのプレイリスト レイアウトを表示.これは、視聴者がすべて見るアプリのボタン。ベストプラクティスとして、ある種のグリッドを選択します。
  7. プレイリストにコンテンツを追加するプロセスを開始するには、資産タブ。選択できるアセットタイプが6つあることがわかります。
    プレイリストのオプションが表示されるところまで来ました。映画、シリーズ、季節、イベントとチャネル、Video Cloud Studio では選択できません。これらのタイプはVideoCloud Studioではアクセスできないため、Brightcove Beaconでプレイリストを作成する必要があります。
  8. 目的の資産を選択するには、まず資産タイプをクリックし、次に 実際の資産の横に表示されます。
