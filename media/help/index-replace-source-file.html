アップロード-ソースの置き換え

{{page.description}}

  1. を選択プロファイルの取り込み.インベストメントプロファイルによって、作成されるレンディションの数と品質が決まります。
  2. 新しいビデオソースを選択するには、次の手順を実行します。
    • ドラッグアンドドロップファイル システムからアップロード モジュールへのファイルまた
    • クリックビデオを参照アップロードする 1 つまたは複数のファイルを参照して選択しますまた
    • クリックURLを入力リモートビデオファイルへのURLを入力します
  3. ファイルの処理が開始されます。ビデオのステータスが処理完了.
