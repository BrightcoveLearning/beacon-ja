操作方法:1つのビデオをWebにパブリッシュする

ステップ

  1. プライマリ ナビゲーションで、メディア.
  2. 公開するビデオの横にあるボックスをオンにします。
  3. クリックWeb に公開します。
  4. の中にプレイヤーを選択ドロップダウン リストで、使用するプレーヤーを選択します。
  5. クリックURLプレビュー プレーヤーの下にある をクリックして、新しいブラウザー タブで開きます。ビデオが表示され、再生できることを確認します。
  6. 埋め込みコードをコピーして Web ページに貼り付けます。
