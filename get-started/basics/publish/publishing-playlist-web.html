前提条件

  • あなたが持っているアップロードした動画あなたの Video Cloud アカウントに
  • を作成しましたマニュアルまた頭いいプレイリスト

ステップ

  1. プライマリ ナビゲーションで、メディア。
  2. 公開するプレイリストを選択:
    • プレイリストがサイド ナビゲーションに表示される場合は、それをクリックしてから をクリックします。プレイリストを公開
    • クリックすべてのプレイリスト左側のナビゲーション メニューでプレイリストを選択し、公開と埋め込み > Web に公開
  3. の中にプレイヤーを選択ドロップダウン リストで、使用するプレーヤーを選択します。
  4. クリックURLプレビュー プレーヤーの下にある をクリックして、新しいブラウザー タブで開きます。プレイリストが表示され、ビデオを再生できることを確認します。
  5. 埋め込みコードをコピーして Web ページに貼り付けます。
