はじめに

登録ユーザとは、Brightcove Beaconアプリケーションでアカウントを作成したユーザのことです。これは、無料のアカウントまたは有料のサブスクリプションです。

登録ユーザに関する詳細データをエクスポートできます。このデータは、Eメールマーケティングキャンペーンや顧客関係プログラムの管理に役立ちます。

エクスポート

登録ユーザの詳細データをCSVファイルとしてエクスポートできます。その後、スプレッドシートプログラムでこのファイルを開き、Eメールマーケティングキャンペーンと顧客関係プログラムを管理できます。

ステップ

  1. Video Cloud Studio で、登録ユーザーモジュール。

  2. クリック書き出す.

  3. 選択する書き出す次のいずれかの場合:（含めるデータのフィルタリングを求めるプロンプトが表示されます）。

    • すべての登録ユーザー- すべての登録ユーザーのデータをエクスポート
    • 新しい利用者- 過去 1 か月以内の新規登録ユーザーのデータをエクスポート

  4. エクスポートファイルに含めるデータを選択します。

    データ範囲

    • データ範囲- エクスポートされたデータの期間を選択します

    データ

    • Eメール- ユーザーのメールアドレス
    • 名前- ユーザー名
    • ユーザー外部 ID - OIDC で認証されたアプリ ユーザーを Beacon アプリ ユーザーに接続する一意のユーザー ID
    • ユーザートークン- ビーコン アプリ ユーザーを指定するための一意のシステム生成ユーザー ID
    • 入社日- ユーザーがアプリにサインアップした日付
    • 前回のログイン- ユーザーがアプリに最後にログインした日付
    • サブスクリプションの詳細- サブスクリプション名、開始日/終了日、ステータス、支払いゲートウェイ、トランザクション ID が含まれます

  5. クリックデータのエクスポート.
