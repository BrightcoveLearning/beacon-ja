ビデオフィールド

ビデオフィールドは、アカウント内の各ビデオに関連付けられたメタデータです。一連の標準フィールドにアクセスでき、独自のカスタムフィールドを作成できます。

カスタムフィールド

ユーザが作成したフィールド。

  • 分野- トグル内部名を表示と表示名を表示カスタムおよび標準フィールド リスト用。
  • 編集するカスタムフィールド名をクリックします。
  • 表示名- カスタム フィールド名。使用できる2バイト文字とスペース。
  • 内部名- API 識別子。小文字、スペースを含まない英数字。
  • タイプ- 次のいずれかを選択します。
    • 文章- テキスト入力フィールド。最大1024文字のシングルバイト文字または512文字のダブルバイト文字。
    • リスト- 値のドロップダウン リスト。各値は最大100文字で、最大150の値です。
  • 説明- フィールド値を設定するためのユーザー指示を提供します。これは、各ビデオの詳細のヒントとして表示されます。メディアページ。
  • このフィールドを必須にする- ビデオを有効にする前に、ユーザーはカスタム フィールド値を入力する必要があります。
カスタムフィールドの追加

独自のカスタムフィールドを作成できます。

  1. クリックカスタム フィールドを追加
  2. カスタムフィールド値を入力します。
  3. クリックフィールドを保存
カスタムフィールドの削除

カスタムフィールドを削除するには、お問い合わせください顧客サポート 。

標準フィールド

明るいカバーがフィールドを提供しました。

  • 編集する標準フィールド名をクリックします。
  • 説明- フィールド値を設定するためのユーザー指示を提供します。これは、各ビデオの詳細のヒントとして表示されます。メディアページ。
  • このフィールドを必須にする- ビデオを有効にする前に、ユーザーはカスタム フィールド値を入力する必要があります。
フィールドを追加する必要がありますか?

各アカウントには、設定された数のカスタムフィールドが与えられます。さらに必要な場合は、お問い合わせください顧客サポート 。
