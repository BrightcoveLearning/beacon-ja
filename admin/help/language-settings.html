言語設定

ビデオメタデータに関連付けられた言語を定義します。

注:使用中の言語を削除すると、その言語に関連付けられているすべてのメタデータは非表示になります。関連付けられたメタデータを復元するには、言語を再度追加します。
プライマリ言語

プライマリ言語は、メディアモジュールでビデオプロパティを編集するときに最初に表示されます。既存のメタデータはプライマリ言語に関連付けられます。

プライマリ言語の編集

  1. 表示されるプライマリ言語をクリックして変更します。
  2. 言語を入力して値のリストを表示します。
  3. 言語を選択します。
  4. クリック保存.
その他の言語

プライマリ言語を保存した後、追加のメタデータ言語を追加できます。これにより、メディアモジュールに多言語メタデータを追加できます。

言語の追加

  1. クリック言語を追加.
  2. の中に言語入力フィールドに言語を入力して値を確認します。
  3. 言語を選択します。
  4. クリック言語を保存.

言語の編集

  1. リスト内の言語をクリックして編集します。
  2. の中に言語入力フィールドに言語を入力して値を確認します。
  3. 言語を選択します。
  4. クリック保存、キャンセル、 また 。
