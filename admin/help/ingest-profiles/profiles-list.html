トランスコードプロファイル

収集プロファイルは、ビデオアップロードごとに作成されるレンダリングの数とタイプを定義します。

表示名を表示 / API 名を表示
クリックして、プロファイル表示名のリストとプロファイル API 名を切り替えます。
フィールドの表示
  • プロファイル名と説明- 名前のリンクをクリックして、プロフィールを編集します
  • プロファイル バージョン(アカウント構成に基づく):
    • 動的配信- レンディションの小さなセットを保存し、リアルタイムのパッケージングを実行します
    • コンテキストアウェア- 機械学習を使用して、必要なレンディションの数と品質を最適化します
  • タイプ:
    • 標準 VOD - ブライトコーブが提供するデフォルトの VOD プロファイル。これは編集または削除できません。
    • カスタム VOD - あなたが作成したカスタム VOD プロファイル
    • スタンダードライブ- ブライトコーブが提供するデフォルトのライブ プロファイル。これは編集または削除できません。
    • カスタムライブ- 作成したカスタム ライブ プロファイル
  • レンディション- 作成できるレンディションの数
  • 画面- アップロードページからプロファイルを非表示/表示に切り替えます
プロファイルの追加
  1. 拡大新しいプロファイル.
  2. 次のいずれかを選択します。
    • 動的配信- 新しい動的配信プロファイルを作成する
    • ライブ- 新しいライブ プロフィールを作成する
プロファイルの削除
  1. クリック 関連付けられた取り込みプロファイルを削除します。
  2. 確認するには、プロフィールを削除.
