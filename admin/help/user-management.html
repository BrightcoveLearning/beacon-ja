ユーザー管理

このアカウントのユーザーを表示、追加、編集、または削除します。

ユーザーリストの表示
  • 名前- ユーザー名と電子メール アドレス。ユーザー名をクリックして、モジュール アクセスを編集します
  • 役割:
    • 標準- ユーザーは管理セクションにアクセスできません
    • 管理者- ユーザーは管理セクションにアクセスできます

    次のいずれかのモジュールへのユーザアクセスを許可します。

  • メディア- ユーザーが動画を編集したり、プレイリストを作成したり、動画を公開したりできるようにします
  • アップロード- ユーザーが Beacon Studio に動画を追加できるようにする
  • 分析- ユーザーがビデオとプレーヤーのトラフィック データを表示できるようにする
  • ライブ- ユーザーがライブ イベントをブロードキャストできるようにする
  • ギャラリー- ユーザーがテンプレートを使用してビデオ コンテンツ ページを作成できるようにする
  • 観客- ユーザーが Video Cloud 分析を Oracle の Eloqua や Marketo などのマーケティング オートメーション プラットフォームに接続できるようにする
  • 社交- ユーザーが YouTube、Facebook、Twitter などのソーシャル メディア プラットフォームに動画を公開できるようにする
  • ページ- ブライトコーブ ビーコン アプリに表示されるコンテンツをユーザーが作成および管理できるようにする
  • 収益化- ユーザーがブライトコーブ ビーコン アプリのサブスクリプション プランを作成および管理できるようにする
  • クラウドプレイアウト- ユーザーが VOD コンテンツを使用してライブ イベントをプログラムおよびスケジュールできるようにする

  • 前回のログイン- ユーザーが最後にログインした日時
ユーザーの追加
  1. クリック+ ユーザーを追加.
  2. 新しいユーザの値を入力します。
    • Eメールアドレス
    • 姓
    • 姓
  3. 役割:
    • 標準- ユーザーは管理セクションにアクセスできません
    • 管理者- ユーザーは管理セクションにアクセスできます
  4. モジュール アクセス- ユーザーがアクセスできるモジュールを選択します。
  5. クリック保存.
ユーザーの編集
  1. 編集するユーザーの行を選択します。
  2. 変更できるのはモジュール アクセスユーザーのために。
  3. クリック保存.
ユーザーの削除
  1. クリック ユーザに関連付けられます。
  2. 確認するには、わかった.
ユーザー数を増やす必要がありますか?

各アカウントには、設定されたユーザ数が与えられます。さらに追加するには、お問い合わせください顧客サポート 。
