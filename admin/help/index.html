アカウント情報

Brightcove Beaconアカウント情報を表示および編集します。

アカウント情報の表示

  • アカウント名- アカウントのわかりやすい名前
  • アカウントID - 割り当てられたアカウント ID
  • 住所- アカウントの郵送先住所
  • 電話- 連絡先の電話番号
  • 最大ユーザー数- アカウントに許可されるユーザーの最大数。でユーザーを管理するユーザー管理セクション
アカウント情報の編集
  1. クリックアカウント情報の編集.
  2. フィールド値を更新します。
    注意:
    • のアカウントID変えられない
    • 最大ユーザー数アカウントの種類に基づいています
  3. クリック保存.
