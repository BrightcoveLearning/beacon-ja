ライブ設定

BrightcoveLiveストリームのフィルタリングプレートと資格情報を管理します。SSAI 広告構成は、サーバー側の広告設定.

塗りつぶしスレートを追加

フィルスレートは、ライブストリーム広告ブレークにギャップがある場合に提供されるコンテンツです。

  1. クリック塗りつぶしスレートを追加.
  2. 入る名前- 塗りつぶしスレートのわかりやすい名前。
  3. 入るソース URL - フィル スレートの完全なアドレス。
  4. クリック保存.
塗りつぶしスレートの削除
  1. クリック 関連付けられたフィルスレートを削除します。
  2. の中に消去ダイアログ、クリック消去確認するために。
資格情報の追加

LiveモジュールまたはLiveAPIを使用する場合は、ライブクレデンシャルが必要です。

  1. Live Video Cloud 資格情報については、 API 認証資格情報 。
    S3、FTP、またはRTMPクレデンシャルの場合は、この手順をスキップします。
  2. 入るラベル- このクレデンシャルの内部名
  3. を選択タイプ:
    • FTP - VOD アセットを ftp の場所に保存するために使用 (API 使用)
    • RTMP - エンコーダーが使用する RTMP クレデンシャルの作成に使用
    • S3 - VOD アセットを Amazon S3 アカウントに保存するために使用 (API 使用)
    • ビデオクラウド- VOD アセットを Video Cloud アカウントに保存するために使用 (API 使用)
  4. 入る公開鍵 / ユーザー名:
    • FTP - FTP サイトのユーザー名を入力します
    • RTMP - エンコーダーが認証に使用するユーザー名を入力します
    • S3 - S3 アカウントのユーザー名を入力します
    • ビデオクラウド- API 認証資格情報用に生成されたクライアント ID を入力します
  5. 入る秘密鍵・パスワード:
    • FTP - FTP サイトのパスワードを入力します
    • RTMP - エンコーダーが認証に使用するパスワードを入力します
    • S3 - S3 アカウントのパスワードを入力します
    • ビデオクラウド- API 認証資格情報用に生成されたクライアント シークレットを入力します。
  6. クリック保存.
資格情報の削除
  1. クリック 関連付けられているクレデンシャルを削除します。
  2. の中に消去ダイアログ、クリック消去確認するために。
